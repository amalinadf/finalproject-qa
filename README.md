# 🚀 OrangeHRM Automation Testing with Cypress

## 📋 Project Overview
This project is an automated testing framework developed for the OrangeHRM web application using Cypress and JavaScript. The framework follows the Page Object Model (POM) design pattern to improve test maintainability, reusability, and scalability.

## 🎯 Objectives
- Automate critical user workflows
- Reduce manual testing effort
- Improve test coverage and efficiency
- Implement a maintainable automation framework

## 🛠️ Tech Stack
- Cypress
- JavaScript
- Git
- GitHub
- Page Object Model (POM)
